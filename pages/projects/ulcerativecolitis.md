# Ulcerative colitis

## ImmUniverse Map

### Description

The ImmUniverse Map is developed as part of IMI2 [ImmUniverse](https://www.immuniverse.eu/) project. It focuses on two diseases: atopic dermatitis and ulcerative colitis.
